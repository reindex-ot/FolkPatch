**FolkPatch** is an extended non-parallel branch developed based on [APatch](https://github.com/bmax121/APatch), which only optimizes interface design and extends features without introducing new core functionalities.

## Key Features
- Built based on MiuiX library
- Kernel-based Android device root solution
- APM: Supports a Magisk-like module system. Supports batch flashing for faster operation
- KPM: Supports modules that inject any code into the kernel (providing kernel function `inline-hook` and `syscall-table-hook`)
- Better customization system, supports custom wallpapers
- Multiple featured languages, switch at will
- Online module download feature
- Monet color extraction feature

## Download & Installation

Download the latest APK from the [Releases page](https://github.com/matsuzaka-yuki/FolkPatch/releases/latest).

## System Requirements

- Supports ARM64 architecture
- Supports Android kernel versions 3.18 - 6.12

## Open Source Information

This project is based on the following open source projects:

- [KernelPatch](https://github.com/bmax121/KernelPatch/) - Core component
- [Magisk](https://github.com/topjohnwu/Magisk) - magiskboot and magiskpolicy
- [KernelSU](https://github.com/tiann/KernelSU) - Application UI and Magisk-like module support
- [APatch](https://github.com/bmax121/APatch) - Upstream branch
